{%trans "Responses"%}

{%trans "Each response is a JSON object."%}

  • {%trans "The data requested is wrapped in the result tag."%}
  • {%trans "If you have a response, it will always be within the result field."%}
  • {%trans "We also include a ok flag and an array of msg in the response."%}
  • {%trans "Some responses can have additional pagination info wrapped in the meta"%}
  • {%trans "An msg object will contain a type field and a text"%}

{%trans "Success Response"%} {%trans "example"%}

{
  "ok": true,
  "result":
  {
      "abc": 123,
      "def": 456
  }
  "msg":
  [
    {
      "type": "info",
      "text": "How are you!"
    }
  ]
}

{%trans "Error Response"%} {%trans "example"%}

{
  "ok": false,
  "msg":
  [
    {
      "type": "error",
      "text": "A verification code was sended to user"
    }
  ]
}
